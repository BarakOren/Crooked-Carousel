__Crooked Carousel__

live demo: https://codesandbox.io/s/caro-31fos?file=/src/carousel.js

how to use:
put your imageURLs inside image.js
"verticalPosition" controls the vertical location of each image.

<Carousel /> Props:

1. id={"carousel1"} // each carousel must have a diffrenet id
2. images={images} // moves the images file to the carousel
3. viewItems={3} // how many images to view in the 100% window width. 2, 3 or 4.
4. carouselHeight={300} // carousel height = px.
5. buttonColor={"white"} // choose arrow color.
